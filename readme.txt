== Description ==

Community theme

== Установка ==

https://support.wpshop.ru/docs/themes/wpcommunity/

Вариант 1.
1. Перейдите в раздел Внешний вид - Темы - Добавить новую - Загрузить тему
2. Выберите скачанный архив и нажмите "Установить"
3. Активируйте тему
4. Перейдите в раздел Настройки - Настройки WPCommunity и введите лицензионный ключ

Вариант 2.
1. Разархивируйте скачанный архив.
2. Поместите папку wpcommunity в /wp-content/themes/
3. Перейдите в раздел Внешний вид - Темы и активируйте тему
4. Перейдите в раздел Настройки - Настройки WPCommunity и введите лицензионный ключ
